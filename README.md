## About
A menu-driven C++ campus management and analytics system developed for PF Final Project. 
This system manages student records using file handling and provides analytics like average GPA, 
attendance tracking, and performance reports.

## Features
1. Register New Student Record
2. Display All Student Records  
3. Search Student by Roll No
4. Update Student Details
5. Delete Student Record
6. Show Analytics Report - Average GPA, Highest CGPA, Low Attendance
7. Exit

## Technologies Used
- Language: C++
- Concepts: Structures, File Handling, Functions, Loops, Arrays
